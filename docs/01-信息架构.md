# POD Dreamworks 信息架构设计

## 产品定位
**一句话定位**：让学生在"真实但安全"的环境中，完成跨境电商全流程实训

**核心价值**：
- 学生：自由创作 + 真实上架 + 成就激励
- 教师：省心管理 + 可控审核 + 数据支撑
- 学校：量化成果 + 风险隔离 + 检查有据

---

## 系统角色

| 角色 | 说明 | 核心场景 |
|------|------|----------|
| 学生 | 电商专业学生 | 学习选品、创作设计、提交上架、追踪订单 |
| 教师 | 实训课教师 | 发布任务、审核作品、批改评分、监控进度 |
| 管理员 | 院系管理者 | 配置学校、管理教师、查看总览报表 |

## 设计规范

### 品牌色彩

```
主色调：
- Primary:     #6366F1 (靛蓝紫) - 主要按钮、强调色
- Primary Dark:#4F46E5          - 悬停状态

功能色：
- Success:     #22C55E (绿色)   - 成功、通过、安全
- Warning:     #F59E0B (橙色)   - 警告、待处理
- Danger:      #EF4444 (红色)   - 错误、危险、退回
- Info:        #3B82F6 (蓝色)   - 信息提示

中性色：
- Dark:        #1E293B          - 主要文字
- Gray:        #64748B          - 次要文字
- Light:       #F1F5F9          - 背景色
- White:       #FFFFFF          - 卡片背景
```

### 角色区分配色

```
学生端：紫色调 - 年轻、活力、创造
- 侧边栏：#1E1B4B → #312E81 渐变
- 强调色：#6366F1

教师端：深蓝调 - 专业、稳重、可信
- 侧边栏：#0F172A → #1E293B 渐变
- 强调色：#3B82F6

管理端：深灰调 - 权威、全局、掌控
- 侧边栏：#111827 → #1F2937 渐变
- 强调色：#6366F1
```
